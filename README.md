# Operava Landing Page

Corporate landing page for Operava, a managed service brand for DevOps, SOC, IT operations, cloud infrastructure, and security monitoring.

## Design System

The design-system source of truth is documented in [OPERAVA_DESIGN_SYSTEM.md](./OPERAVA_DESIGN_SYSTEM.md).

Core rules:

- Light-mode corporate interface
- Inter typography with max `600` font weight
- Heading line-height `160%`, body line-height `150%`
- 8px spacing system
- Square buttons with `0px` radius
- White cards with subtle borders and soft shadows
